# Infinity Hub - متجر الملابس الرجالية

متجر إلكتروني متكامل لبيع الملابس الرجالية في مصر مع الدفع عند الاستلام.

## المميزات

- ✅ عرض المنتجات مع صورها وأسعارها
- ✅ سلة تسوق تفاعلية
- ✅ اختيار المحافظة وحساب تكلفة الشحن تلقائياً
- ✅ الدفع عند الاستلام
- ✅ الشحن لجميع محافظات مصر
- ✅ لوحة تحكم للمالك لإدارة الطلبات
- ✅ إمكانية تعديل أسعار الشحن لكل محافظة من لوحة التحكم
- ✅ إضافة منتجات جديدة
- ✅ تتبع حالة الطلبات (قيد المعالجة، مؤكد، تم الشحن، تم التسليم)

## التثبيت والتشغيل

### المتطلبات
- Node.js (الإصدار 14 أو أحدث)
- npm

### خطوات التشغيل

1. انتقل إلى مجلد المشروع:
```bash
cd infinity_hub
```

2. ثبت المكتبات المطلوبة:
```bash
npm install
```

3. شغل الخادم:
```bash
npm start
```

4. افتح المتصفح وانتقل إلى:
- **المتجر الرئيسي**: http://localhost:3000
- **لوحة التحكم**: http://localhost:3000/admin

## كيفية الاستخدام

### للعملاء:
1. تصفح المنتجات في الصفحة الرئيسية
2. أضف المنتجات للسلة
3. اختر المحافظة لحساب تكلفة الشحن
4. أدخل بياناتك (الاسم، الهاتف، العنوان)
5. اضغط على "إتمام الطلب"
6. ادفع عند استلام الطلب

### لمالك المتجر:
1. انتقل إلى لوحة التحكم: http://localhost:3000/admin
2. **إدارة الطلبات**: عرض جميع الطلبات وتغيير حالتها
3. **تعديل أسعار الشحن**: عدل سعر الشحن لكل محافظة حسب الحاجة
4. **إضافة منتجات**: أضف منتجات جديدة للمتجر

## المحافظات وأسعار الشحن الافتراضية

تم تضمين جميع محافظات مصر بأسعار شحن افتراضية يمكن تعديلها من لوحة التحكم:

## التقنيات المستخدمة

- **Backend**: Node.js, Express
- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Template Engine**: EJS
- **Data Storage**: JSON file (يمكن تطويره لاستخدام قاعدة بيانات)
